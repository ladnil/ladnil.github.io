[ folio ]

Terminal Application: myMDb

Made for the modern and minimalist cinephile, myMDb is a Terminal app built in Ruby with a simple and fun use. Within myMDb's current scope as a minimum viable product (MVP), its primary purpose for users is simple: search, suggest, save. Users can easily find, display and add movie titles to their watchlist, or have the application suggest one based on their mood/genre. It relies on two tty Ruby gems and data from oMDB’s API to return on a suggestion on an interface with sporadic animations throughout.
